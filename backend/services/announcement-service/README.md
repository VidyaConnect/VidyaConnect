# Announcement Service

## Overview

The Announcement Service manages announcements shared within the VidyaConnect platform.

## Responsibilities

- Create announcements
- Update announcements
- Retrieve announcements
- Manage school announcements
- Support communication workflows

## Technology Stack

- Node.js
- Express.js
- Docker
- REST API
