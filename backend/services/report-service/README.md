# Report Service

## Overview

The Report Service generates academic and administrative reports for the VidyaConnect platform.

## Responsibilities

- Generate reports
- Provide reporting APIs
- Process academic data
- Support administrative reporting
- Manage report generation logic

## Technology Stack

- Node.js
- Express.js
- Docker
- REST API
