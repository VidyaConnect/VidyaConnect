# Assignment Service

## Overview

The Assignment Service manages assignments created by teachers and accessed by students within the VidyaConnect platform.

## Responsibilities

- Create and manage assignments
- Store assignment details
- Provide assignment-related APIs
- Support teacher and student assignment workflows
- Manage assignment business logic

## Technology Stack

- Node.js
- Express.js
- Docker
- REST API
