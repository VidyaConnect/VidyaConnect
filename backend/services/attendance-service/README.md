# Attendance Service

## Overview

The Attendance Service manages student attendance records within the VidyaConnect platform.

## Responsibilities

- Record student attendance
- Retrieve attendance information
- Manage attendance records
- Provide attendance APIs
- Support teacher attendance workflows

## Technology Stack

- Node.js
- Express.js
- Docker
- REST API
