# File Service

## Overview

The File Service manages file upload, storage, and retrieval operations in VidyaConnect.

## Responsibilities

- Handle file uploads
- Manage file metadata
- Provide file access APIs
- Integrate with cloud storage
- Manage stored files

## Technology Stack

- Node.js
- Express.js
- Docker
- REST API
