# Notification Service

## Overview

The Notification Service manages system notifications and communication events within VidyaConnect.

## Responsibilities

- Send notifications
- Manage notification events
- Support email notifications
- Support push notifications
- Provide notification APIs

## Technology Stack

- Node.js
- Express.js
- Docker
- REST API
