VidyaConnect Container Diagram

Overview

This folder contains the C4 Container Diagram of the VidyaConnect system.

The diagram illustrates the major software containers that make up the VidyaConnect platform and how they communicate with each other.

The diagram includes:

Mobile Application (React Native)
Web Admin Portal (Next.js)
Backend REST API (Node.js & Express.js)
PostgreSQL Database
AWS S3 File Storage
AWS SNS Notification Service
AWS CloudWatch Monitoring
External Users and System Interactions

It also shows the communication between the client applications, backend services, database, cloud services, and external infrastructure.
