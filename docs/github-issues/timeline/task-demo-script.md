# Task: Prepare Mid-September Demo Script

## Goal

Create a repeatable demo script for the university review.

## Recommended Demo Flow

1. Start local environment using Docker Compose.
2. Show Keycloak login.
3. Log in as School Admin/Teacher.
4. Create an announcement.
5. Log in as Parent/Student.
6. View announcement.
7. Log in as Teacher.
8. Mark attendance.
9. Show parent absence view.
10. Submit absence reason and upload document.
11. Show LocalStack-backed file metadata.
12. Show logs/health/Grafana dashboard.
13. Show UI screens for remaining modules.

## Acceptance Criteria

- Demo script exists.
- Demo data is prepared.
- Each step has expected result.
- One student can run the demo while others explain architecture and product value.

`task`, `demo`, `presentation`
