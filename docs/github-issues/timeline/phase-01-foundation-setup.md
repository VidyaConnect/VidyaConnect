# Phase 1: Foundation Setup

## Target Dates

2026-07-09 to 2026-07-21

## Goal

Create the technical foundation required for implementation: local environment, service skeletons, Keycloak, PostgreSQL, and API contracts.

## Tasks

- Finalize corrected architecture and technical direction docs.
- Create API contracts v1.
- Set up initial repo structure.
- Set up local Docker Compose.
- Add PostgreSQL local container.
- Add Keycloak local container.
- Add Nginx local container.
- Add LocalStack local container.
- Create backend microservice skeletons.
- Add `/health` endpoint per service.
- Create initial web/admin UI shell.
- Create initial mobile app navigation shell.

## Acceptance Criteria

- Team can start the local environment.
- Keycloak is accessible locally.
- PostgreSQL is accessible locally.
- At least one backend service runs locally.
- API contracts v1 exist in Markdown.
- Web/admin and mobile UI shells are visible.
